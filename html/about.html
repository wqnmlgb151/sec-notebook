关于 sec-notebook

以攻促防，构建 Web 安全知识体系

项目背景

Web 应用已成为现代社会的基础设施，但安全问题层出不穷。 从 OWASP 每年发布的 Top 10 榜单可以看出，SQL 注入、XSS、CSRF 等经典漏洞在数十年后依然活跃。本项目旨在为前端开发者、安全初学者 提供一个系统化的 Web 安全学习平台，从攻击者视角理解漏洞原理， 从根本上建立安全防御思维。

项目目标

📚

知识体系

基于 OWASP 标准分类，覆盖主流 Web 漏洞类型

🔬

实战导向

每个漏洞配有攻击示例和防御方案

🛡️

防御优先

强调安全编码实践和防护措施

🎓

教育目的

所有内容仅用于学习，不涉及真实攻击

技术栈

技术 版本 用途
HTML5 - 语义化页面结构，Header/Nav/Main/Footer 四部分布局
CSS3 - 响应式布局(Flex+Grid)、玻璃态设计、动画特效
JavaScript (ES6+) - 导航交互、表单验证、Toast 系统、滚动动画
Vue.js 3.x 漏洞库搜索筛选、数据驱动视图、详情弹窗
响应式设计 - 三端适配: ≤768px手机 / 769-1024px平板 / ≥1025px桌面

设计理念

本项目的 UI 设计采用 Dark Cyber Glassmorphism 风格，融合以下设计元素：

暗色主题 玻璃态/毛玻璃 霓虹辉光 噪点纹理 大圆角卡片 终端风格 Glitch动画 扫描线效果 Bento Grid 响应式三端适配

开发规划

第一阶段：主题与功能确定

确定网络安全渗透测试主题，规划核心功能清单与页面结构，完成需求分析文档

第二阶段：基础页面搭建

完成 HTML5 语义化结构、CSS3 暗色主题与玻璃态样式系统、基础页面布局

第三阶段：交互与响应式

JavaScript 导航交互、Vue3 漏洞库模块、三端响应式适配、表单验证系统

第四阶段：测试与文档

兼容性测试、功能测试、撰写课程作品说明书、准备答辩材料
